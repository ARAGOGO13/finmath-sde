# SDE Models — Numerical Simulation of Stochastic Differential Equations

Реализация численного моделирования стохастических дифференциальных уравнений из приложения C книги **Brigo & Mercurio "Interest Rate Models — Theory and Practice"**.

Проект охватывает разделы C.1–C.6: броуновское движение, схемы дискретизации Эйлера и Мильштейна, модели GBM / Vasicek / CIR, процесс Пуассона.

---

## Быстрый старт

### Зависимости

Требуется только:
- **CMake** ≥ 3.16
- **C++17** компилятор (GCC, Clang, MSVC)
- **Git** (для скачивания Matplot++ при сборке)

На macOS дополнительно нужен gnuplot для отрисовки графиков:
```bash
brew install gnuplot
```

На Ubuntu/Debian:
```bash
sudo apt install gnuplot
```

Matplot++ скачается и соберётся **автоматически** через CMake FetchContent при первой сборке — ничего устанавливать вручную не нужно.

---

### Сборка и запуск

```bash
git clone https://github.com/YOUR_USERNAME/sde_models.git
cd sde_models

mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . --parallel

./sde_models
```

Графики сохраняются в папку `plots/` рядом с бинарником.

> **Первая сборка** занимает 3–5 минут — идёт компиляция Matplot++.  
> Повторные сборки быстрые.

## Что демонстрирует проект

### C.1 — Броуновское движение
Траектории непрерывны, но нигде не дифференцируемы: приращение `dW ~ sqrt(dt)`, поэтому `dW/dt ~ 1/sqrt(dt) → ∞`.

### C.3 — Схемы Эйлера и Мильштейна
PDF утверждает что Мильштейн точнее — проект это **измеряет**:

```
Схема        Порядок сходимости   Наклон log-log (замеренный)
-----------  -------------------  ---------------------------
Эйлер        O(sqrt(dt))          ~0.50
Мильштейн    O(dt)                ~1.00
```

Поправка Мильштейна: `0.5 * σ * σ'(x) * (dW² - dt)`

При `σ'(x) = 0` (Vasicek) — схемы совпадают, что подтверждается численно.

### C.4 — Три модели процентной ставки

| Модель | СДУ | X < 0? | Мильштейн ≠ Эйлер? |
|--------|-----|--------|---------------------|
| GBM | dX = μX dt + σX dW | нет (лог-норм.) | да |
| Vasicek | dX = θ(μ-X)dt + σ dW | возможно | нет |
| CIR | dX = θ(μ-X)dt + σ√X dW | нет (при усл. Феллера) | да |

**Нетривиальный эффект GBM**: из-за поправки Ито среднее и медиана расходятся:
```
E[X_t]      = X₀ · exp(μt)               — среднее
median(X_t) = X₀ · exp((μ - σ²/2)t)      — медиана (медленнее!)
```

### C.6 — Процесс Пуассона
Верификация: `E[N_T] = Var[N_T] = λT` — характеристическое свойство распределения Пуассона.

## Связь с практикой

- **GBM** → ценообразование опционов (модель Блэка–Шоулса)
- **Vasicek / CIR** → модели краткосрочной процентной ставки
- **Процесс Пуассона** → моделирование момента дефолта в intensity-based моделях кредитного риска (Cox process / процесс Кокса)
- **Схема Мильштейна** → точное ценообразование path-dependent деривативов
