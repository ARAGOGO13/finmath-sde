# SDE Models — Numerical Simulation of Stochastic Differential Equations

Реализация численного моделирования стохастических дифференциальных уравнений из приложения C книги **Brigo & Mercurio "Interest Rate Models — Theory and Practice"** с дополнительными финансовыми экспериментами (теорема Гирсанова, дискретное дельта-хеджирование).

Проект демонстрирует:
- численные схемы Эйлера и Мильштейна,
- модели GBM, Vasicek (OU), CIR, процесс Пуассона,
- отношение коррелированных GBM,
- проверку теоремы Гирсанова на трёх оценщиках цены колла,
- анализ дискретного дельта-хеджирования короткой позиции в опционе.

## Быстрый старт

### Зависимости

- **CMake** ≥ 3.16
- **C++17** компилятор (GCC, Clang, MSVC)
- **Git** (для автоматической загрузки Matplot++)

Для отрисовки графиков необходим **gnuplot**:

```bash
# macOS
brew install gnuplot

# Ubuntu/Debian
sudo apt install gnuplot
```

Библиотека Matplot++ скачивается и собирается автоматически через CMake `FetchContent` — устанавливать её вручную не требуется.

### Сборка и запуск

```bash
git clone https://github.com/YOUR_USERNAME/sde_models.git
cd sde_models

mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . --parallel

./sde_models
```

Все графики сохранятся в папку `plots/` в корне проекта.

> **Первая сборка** занимает несколько минут — компилируется Matplot++.  
> Повторные сборки быстрые.

## Что демонстрирует проект

### C.3 — Сильная сходимость Эйлера и Мильштейна

Измеряется средняя абсолютная ошибка `E[|X_exact − X_scheme|]` для GBM.
В log‑log масштабе наклоны подтверждают теоретические порядки:

| Схема     | Порядок сходимости | Измеренный наклон |
|-----------|--------------------|-------------------|
| Эйлер     | O(√Δt)             | ≈0.50             |
| Мильштейн | O(Δt)              | ≈1.00             |

### C.4 — Процессы процентных ставок

| Модель   | СДУ                              | X < 0?            | Мильштейн ≠ Эйлер? |
|----------|----------------------------------|-------------------|--------------------|
| GBM      | dX = μX dt + σX dW               | нет (логнормальн.) | да                 |
| Vasicek  | dX = θ(μ‑X) dt + σ dW            | возможно          | нет                |
| CIR      | dX = θ(μ‑X) dt + σ√X dW          | нет при усл. Феллера | да              |

**GBM**: среднее `E[X] = X₀·exp(μt)`, медиана — `X₀·exp((μ−σ²/2)t)`.  
Расхождение вызвано поправкой Ито.

### C.6 — Процесс Пуассона

Проверяется фундаментальное свойство: `E[N_T] = Var[N_T] = λ·T`.

## Финансовые приложения

### G1 — Проверка теоремы Гирсанова

Цена европейского колла вычисляется тремя методами:

1. Аналитическая формула Блэка‑Шоулза.
2. Монте‑Карло под риск‑нейтральной мерой Q (дрейф `r`).
3. Монте‑Карло под физической мерой P (дрейф `μ`) с весом Радона‑Никодима `dQ/dP`.

Все три оценки совпадают в пределах статистической погрешности — численная иллюстрация теоремы Гирсанова.  
При этом отношение дисперсий `Var(P‑MC) / Var(Q‑MC)` экспоненциально зависит от `λ²·T`, что наглядно показывает «стоимость смены меры» и обосновывает использование риск‑нейтральной меры в финансах.

График: `G1_girsanov_merged.pdf` (2 страницы).

### G2 — Дискретное дельта‑хеджирование

Продавец колла получает премию `BS(σ_impl)` и ребалансирует портфель `N` раз, поддерживая дельта‑нейтральность.  
Итоговый P&L моделируется при разных сценариях.

Три подэксперимента:

| Эксперимент | Результат |
|-------------|-----------|
| Гистограммы P&L для N ∈ {10, 50, 250, 1000} | Распределение сужается к нулю с ростом N |
| log(std(P&L)) от log(Δt) | Наклон ≈ +0.5 (Bertsimas–Kogan–Lo 1998) |
| Зависимость среднего P&L от σ_real (σ_impl=0.20) | При σ_real > σ_impl продавец в среднем теряет; теоретическая кривая: `0.5·Γ·S²·(σ_impl²−σ_real²)·T` |

График: `G2_hedging_merged.pdf` (3 страницы).

## Связь с практикой

- **GBM** → ценообразование опционов (Блэк–Шоулз).
- **Vasicek / CIR** → модели краткосрочной процентной ставки.
- **Процесс Пуассона** → моделирование момента дефолта в intensity‑based моделях кредитного риска.
- **Схема Мильштейна** → точное ценообразование path‑dependent деривативов.
